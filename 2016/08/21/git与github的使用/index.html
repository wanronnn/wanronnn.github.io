#git与github的使用

##git-控制版本

###基本命令

  • git init 初始化一个版本库
  • git add . 或者 git add –all 跟踪所有文件
  • git commit -m “注释” 提交文件 设置一个节点 并且注释
  • git log 提交日志
  • git remote add origin url 添加远程版本库
  • git push -u origin master 推送master分支所有内容
    ###分支
  • git branch 查看分支
  • git branch nanme-分支名 创建分支
  • git checkout name 切换到name分支
    ###个人信息
  • git config –global user.name “” 注意之间的空格
  • gir config –global user.name “”
  • git config –list 个人信息列表
    ###.shh
  • ssh-keygen -t rsa -C “your@xxx.com” 创建ssh key 在用户主目录下
  • 公钥：id_rsa.pub
  • 私钥：id_rsa
    • 把公钥复制到github设置里面的ssh key里

##github-在线托管

  • 把项目以网页的形式呈现：
    • 建立gh-pages分支。”账户.github.io/项目名”
  • 建立github个人页面。
