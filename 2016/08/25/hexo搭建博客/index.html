部署到github

  • 生成静态网页
    • hexo g 生成静态网页默认文件夹public
  • 安装git工具
    • npm install hexo-deplouer-git –save

  • 部署到github

    • hexo d

    部署到github就是在github建一个wanronnn.github.io的仓库，然后把public里的静态网页上传到这个仓库里。
    然后，直接在地址栏输入wanronn.github.io
