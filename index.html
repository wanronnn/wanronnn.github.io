hexo搭建博客

用hexo搭建个静态博客

建站

  • 安装node.js 配置环境变量
  • 安装hexo
    npm installl -g hexo-cli
  • 创建所需文件
    hexo init  <文件夹>
cd <文件夹>
npm install

    配置信息

  • 上传到github所需配置
    url: http://wanronnn.github.io
root: /
type: git
repo: git@github.com:wanronnn/wanronnn.github.io.git
branch: master

    部署到github

  • 生成静态网页
    • hexo g 生成静态网页默认文件夹public
  • 安装git工具
    • npm install hexo-deplouer-git –save

  • 部署到github

    • hexo d

    部署到github就是在github建一个wanronnn.github.io的仓库，然后把public里的静态网页上传到这个仓库里。然后，直接在地址栏输入wanronn.github.io

git与github的使用

#git与github的使用

##git-控制版本

###基本命令

  • git init 初始化一个版本库
  • git add . 或者 git add –all 跟踪所有文件
  • git commit -m “注释” 提交文件 设置一个节点 并且注释
  • git log 提交日志
  • git remote add origin url 添加远程版本库
  • git push -u origin master 推送master分支所有内容
    ###分支
  • git branch 查看分支
  • git branch nanme-分支名 创建分支
  • git checkout name 切换到name分支
    ###个人信息
  • git config –global user.name “” 注意之间的空格
  • gir config –global user.name “”
  • git config –list 个人信息列表
    ###.shh
  • ssh-keygen -t rsa -C “your@xxx.com” 创建ssh key 在用户主目录下
  • 公钥：id_rsa.pub
  • 私钥：id_rsa
    • 把公钥复制到github设置里面的ssh key里

##github-在线托管

  • 把项目以网页的形式呈现：
    • 建立gh-pages分支。”账户.github.io/项目名”
  • 建立github个人页面。
    • 创建：wanronnn.github.io
    • 浏览：wanronnn.github.io
